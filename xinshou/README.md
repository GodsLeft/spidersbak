# 天池新手赛
- 统计每天三种行为的数量，做图
- 统计总人数
- 统计商品总数
- 统计每个用户三种行为总次数

## 思路
分割数据，分离出最后一天的数据
编写求正确率和f值的函数

## 运行流程
    trainD2P.py:     用户行为，根据商品子集
    extglobalFea.py：抽取用户全局信息
    extlocalFea.py:  抽取用户局部信息
    extItemlclFea.py:抽取商品局部信息
    cbFea.py:        结合商品特征
    labelFea.py:     特征向量打上标签
    classification.py: 分类模型
    split_.py        转换成提交文件
